# Rakuten Products Management Solution
Rest API for managing products &amp; categories in a secured context

## Guide

## Prerequisites
- JDK 1.8 or later
- Maven 3 or later
- MySQL 5.6 or later

## Stack
- Spring MVC
- Spring Security
- Spring Data JPA
- Maven
- MySQL

## Run
Deploy on any web server, run https://localhost:8443/rakuten-ecommerce/swagger-ui.html
